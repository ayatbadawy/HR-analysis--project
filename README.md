👥 HR Analytics Dashboard Project
📝 Overview
This project is a Human Resources (HR) Data Analysis Dashboard built using Power BI.
It aims to analyze employee data to support HR decision-making, improve workforce management, and track key HR metrics.
🎯 Objectives
Analyze employee distribution across departments and roles
Track hiring, attrition, and retention rates
Monitor employee performance and satisfaction
Identify trends in workforce demographics
Support strategic HR decisions using data insights
🛠️ Tools Used
Power BI
Excel (Data Cleaning & Preparation)
Data Modeling
DAX Measures (if applicable)
📊 Dashboard Features
Interactive filters (Department, Gender, Job Role, etc.)
KPI cards (Total Employees, Attrition Rate, Active Employees, etc.)
Charts for employee distribution and turnover analysis
Performance and satisfaction analysis
Clean and interactive visuals for better HR insights
🔍 Key Insights
Certain departments have higher employee turnover rates
Employee distribution varies across job roles and levels
Attrition trends can be linked to specific factors such as satisfaction or workload
Data helps improve hiring and retention strategies
📁 Files Included
Power BI Dashboard (.pbix)
Dataset (Excel file)
README file
🚀 Conclusion
This project demonstrates how HR analytics using Power BI can help organizations better understand their workforce, reduce attrition, and improve employee management through data-driven insights.
